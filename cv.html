Projects

Selected technical projects

A380 Flight Performance Tool

Development of an open-source aviation performance application initially focused on Airbus A380 take-off performance calculations.

The project includes aircraft performance logic, runway and weather-data integration, take-off configuration modelling, operational-limit checks and a structured performance calculation interface.

The work involves interpreting and organising aircraft, atmospheric, runway and operational data for use within the application.

Technologies: TypeScript, React, Vite, Git, GitHub and aviation performance-data analysis.

Status: active development.
