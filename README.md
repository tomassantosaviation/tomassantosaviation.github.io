# Tomás Santos Aviation

Personal aviation portfolio website for Tomás Santos.

Website: https://tomassantosaviation.com

## About

This repository contains the source code and public content for my personal aviation website. The website is used to present my aviation background, technical projects, safety reports, and learning work as I prepare for professional pilot training.

The site includes aviation-focused writing, project documentation, and selected technical work related to flight simulation, aircraft performance, flight training, and aviation safety.

## Sections

The website may include:

* Aviation safety reports and articles
* Technical aviation projects
* Flight simulation and performance-calculation work
* Pilot-training notes and portfolio material
* Contact and professional profile information

## Important Notice

All aviation material published on this website is for educational, portfolio, and personal research purposes only.

Nothing on this website should be used for real-world flight planning, aircraft operation, dispatch, training-device qualification, regulatory compliance, or operational decision-making. Approved manuals, official publications, instructor guidance, operator procedures, air traffic control instructions, and applicable aviation regulations always take precedence.
